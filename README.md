# Dotfiles

## Installation

```sh
$ curl -sL -o --create-dirs ~/.config/chezmoi/chezmoi.toml "https://raw.githubusercontent.com/mt-coff/dotfiles/master/dot_config/chezmoi/chezmoi.toml"
$ sh -c "$(curl -fsLS chezmoi.io/get)" -- init --apply mt-coff
```
